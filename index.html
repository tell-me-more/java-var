Variablen verstehen und in Java anwenden

Wie merkt sich ein Programm einen Namen, ein Alter oder einen Punktestand?

Start / Einstieg

Stell dir vor, ein Programm hat viele beschriftete Boxen. Auf jeder Box steht ein Name (z. B. punkte), und in der Box liegt ein Wert (z. B. 0). So arbeitet eine Variable.

Was ist eine Variable?

Eine Variable ist ein benannter Speicherplatz für einen Wert. Der Name bleibt gleich, der Inhalt kann sich ändern.

Interaktiv: Variablen-Box

Name: punkte
Wert: 0

Beobachte: Der Name bleibt „punkte“, nur der Wert ändert sich.

Variablen im Algorithmus

Algorithmen brauchen Variablen für:

  • Eingaben (z. B. Name, Alter)
  • Zwischenergebnisse (z. B. Punkte zählen)
  • Ausgaben (z. B. Ergebnis anzeigen)

Pseudocode (einfach)

name ← "Tom"
alter ← 12
punkte ← 0
punkte ← punkte + 1
Ausgabe(name, alter, punkte)

In Java

String name = "Tom";
int alter = 12;
int punkte = 0;
punkte = punkte + 1;

Variablen in Java

Wichtige Syntax:

  • Datentyp name; → Deklaration
  • Datentyp name = wert; → Initialisierung
  • name = neuerWert; → Veränderung
String name = "Tom";
int alter = 12;
boolean heuteSchule = true;
double preis = 2.99;
int punkte = 0;
punkte = punkte + 1;

Schritt-für-Schritt: punkte = punkte + 1;

Klicke auf „Nächster Schritt“ und verfolge, was passiert.

    Datentypen

    In Java hat jede Variable einen Typ.

    int

    Ganze Zahlen, z. B. 12

    double

    Kommazahlen, z. B. 3.5 oder 2.99

    boolean

    Wahrheitswerte: true oder false

    String

    Text in Anführungszeichen, z. B. "Mia"

    Übung: Welcher Datentyp passt?

    Häufige Fehler

    • Eine Variable ist nicht der Wert selbst.
    • = bedeutet in Java Zuweisung.
    • String-Werte brauchen Anführungszeichen.
    • Java unterscheidet Groß- und Kleinschreibung.

    Übung: Fehler finden

    int alter = 12;

    String name = Mia;

    boolean fertig = "true";

    Merkkasten / Zusammenfassung

    Eine Variable ist ein benannter Speicherplatz für einen Wert. In Java hat jede Variable einen Typ, einen Namen und oft einen Startwert.

    • Denke an eine beschriftete Box.
    • Wähle klare Namen, z. B. alter statt a.
    • Beachte den Typ: int, double, boolean, String.
