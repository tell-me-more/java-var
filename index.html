Variablen verstehen und in Java anwenden

Ein Programme arbeitet oft mit Daten: mit Punkten in einem Spiel, dem Alter einer Person oder dem Ergebnis einer Rechnung.
Statt alles immer neu einzugeben, speichert man solche Werte in Variablen.

Einstieg

Stell dir vor, ein Programm hat viele beschriftete Boxen. Auf jeder Box steht ein Name (z. B. punkte, alter, lieblingsfach, usw.), und in der Box liegt ein Wert (z. B. 0, 15, Informatik). So arbeitet eine Variable.

Was ist eine Variable?

Eine Variable ist ein benannter Speicherplatz für einen Wert. Der Name bleibt gleich, der Inhalt kann sich aber ändern. Über den Namen kann ich immer wieder auf diesen Speicherplatz zugreifen und den aktuellen Inhalt abrufen, verändern oder ausgeben.

Variablen-Box

Name: punkte
Wert: 0

Beobachte: Der Name bleibt „punkte“, nur der in der Variablen "punkte" gespeicherte Wert ändert sich.

Einsatz von Variablen

Variablen werden für viele verschiedene Dinge benötigt. Algorithmen brauchen Variablen für:

  • Eingaben (z. B. Name, Alter)
  • Zwischenergebnisse (z. B. Punkte zählen)
  • Ausgaben (z. B. Ergebnis anzeigen)

Pseudocode

name ← "Tom"
alter ← 12
punkte ← 0
punkte ← punkte + 1
Ausgabe(name, alter, punkte)

In Java

String name = "Tom";
int alter = 12;
int punkte = 0;
punkte = punkte + 1;
println(name, alter, punkte);

Variablen in Java

In Java musst du dich an bestimmte Vorgaben halten, wie Variablen notiert werden.

Diese Vorgaben nennt man Syntax. Wichtig ist außerdem: In Java endet jede Code-Zeile mit einem Semikolon.

  • Datentyp name; → Deklaration
  • Datentyp name = wert; → Initialisierung
  • name = neuerWert; → Veränderung
String name = "Tom";
int alter = 12;
boolean heuteSchule = true;
double preis = 2.99;
int punkte = 0;
punkte = punkte + 1;

Schritt-für-Schritt:
int punkte = 0
punkte = punkte + 1;

Klicke auf „Nächster Schritt“ um zu verstehen, was die beiden Codezeilen genau machen.

    Datentypen

    In Java hat jede Variable einen Typ.

    int

    Ganze Zahlen, z. B. 12

    double

    Kommazahlen, z. B. 3.5 oder 2.99

    boolean

    Wahrheitswerte: true oder false

    String

    Text in Anführungszeichen, z. B. "Mia"

    Übung: Welcher Datentyp passt?

    Häufige Fehler

    • Eine Variable ist nicht der Wert selbst.
    • = bedeutet in Java Zuweisung (keine mathematische Gleichheit).
    • Wenn du zwei Werte/Variablen auf Gleichheit überprüfen willst, dann nutze ==
    • String-Werte brauchen Anführungszeichen.
    • Java unterscheidet streng zwischenGroß- und Kleinschreibung.

    Merkkasten

    Eine Variable ist ein benannter Speicherplatz für einen Wert. In Java hat jede Variable einen Typ, einen Namen und oft einen Startwert.

    • Denke an eine beschriftete Box.
    • Wähle klare Namen, z. B. alter statt a. So weißt du immer, was in der Box drin ist.
    • Beachte den Typ: int, double, boolean, String.
